Overview

Package minify provides support for minifying web assets in preparation for deployment. This package is made possible by the excellent contributions to the minify project. If you find this useful, please check out their repo. https://github.com/tdewolff/minify

ClearBeforeWrite [link]

func ClearBeforeWrite() ConfigFunc

ClearBeforeWrite will make this Minifier clear the asset directory before writing any new files.

HashDigits [link]

func HashDigits(digits int) ConfigFunc

HashDigits sets the number of hash digits to use for minified files. Digits must be between 4-32, inclusive.

New [link]

func New(mappingFile mm.PathString, assetDirName string, configFuncs ...ConfigFunc) (*Minifier, error)

New is used to create a new Minifier. It allows specifying policies for minifying web assets.

PackageName [link]

func PackageName(packageName string) ConfigFunc

PackageName sets the package name used in the mapping file. The default when no package is specified is the parent directory name.

Version [link]

func Version(version string) ConfigFunc

Version sets the version of minify to use. Defaults to "latest".

Minifier [link]

type Minifier struct {
	// Has unexported fields.
}

Minifier defines consistent policies for minifying web assets. The type is immutable after creation with New.

Minifier.Apply [link]

func (mini *Minifier) Apply(b *mm.Build)

Apply will link Minifier operations to the 'generate' step of a Modmake Build. Further operations may be added to the Minifier after calling Apply, but before the generate step is ran.

Minifier.MapCSSBundle [link]

func (mini *Minifier) MapCSSBundle(bundleName string, sources ...mm.PathString) *Minifier

MapCSSBundle will attempt to bundle one or more CSS files into one bundle file, and add embed entries into the configured mapping file. Source files should have the ".css" file extension.

Minifier.MapFile [link]

func (mini *Minifier) MapFile(source mm.PathString) *Minifier

MapFile will attempt to minify the given file and add embed entries into the configured mapping file.

Minifier.MapJSBundle [link]

func (mini *Minifier) MapJSBundle(bundleName string, sources ...mm.PathString) *Minifier

MapJSBundle will attempt to bundle one or more JS files into one bundle file, and add embed entries into the configured mapping file. Source files should have the ".js" file extension.
